Create New Project

Welcome to the project creation page! Here, you’ll give your project a name and a brief description, and pick the number of generating stations in your model. The model setup is meant to be flexible, it is possible to simulate a single turbine or multiple turbines at multiple facilities. Once you are satistifed with your choice of simulation mode, click “Save Project,” and you’ll move on to defining your flow scenarios and other details. If you need help with any field, look for the little info icons or contact support via GitHub. If at any point in time you want to start over, simpy navigate to the home page and log out. Any session data you create will be destroyed.
